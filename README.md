# National-morbidity-mortality-and-air-pollution-study
# Bayesian Linear Regression with INLA (NMMAPS Data)
Overview

This project demonstrates Bayesian linear regression using the INLA framework in R.
The analysis models the relationship between PM10 air pollution levels and temperature using data from the NMMAPS dataset.
<img width="399" height="242" alt="Rplot02" src="https://github.com/user-attachments/assets/53b3753a-7fab-45fc-8f67-468b995236f5" />


The goal is to:

Fit a Gaussian linear regression model

Explore different prior specifications

Visualize posterior distributions
